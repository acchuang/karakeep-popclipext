# Karakeep PopClip Extension

Save selected URLs to your [Karakeep](https://karakeep.app/) instance directly from [PopClip](https://pilotmoon.com/popclip/) on macOS.

## Install

Download the latest `Karakeep.popclipext.zip` from [Releases](https://github.com/acchuang/karakeep-popclipext/releases), unzip it, then double-click `Karakeep.popclipext` and confirm the install in PopClip.

## Setup

After installing, open the extension's options in PopClip preferences and set:

- **Instance URL** — e.g. `https://karakeep.example.com`
- **API Key** — from Karakeep → Settings → API Keys

## Actions

- **Save to Karakeep** — select a URL, then click the bookmark icon to save it as a bookmark.
- **Test Karakeep Connection** — verifies the Instance URL and API Key are correct.

## Requirements

- macOS with [PopClip](https://pilotmoon.com/popclip/) installed
- A Karakeep instance with API access
